# test-repo-graphql-tag

## Before upgrading

```sh
git clone https://github.com/alves-omedetaro/test-repo-graphql-tag/tree/main
cd test-repo-graphql-tag
yarn

yarn upgrade-interactive
```

```sh
git checkout feature/upgrade-graphql-tag
yarn
yarn upgrade-interactive
```
